Echolume Privacy

Privacy Policy

Last updated: May 2026

Overview

Echolume values your privacy. Echolume is a macOS application designed to turn live audio input into real-time visual renders. All processing runs locally under the strict security parameters of the macOS Application Sandbox. This document describes our policies regarding data collection, local processing, and external reporting.

Local audio processing

Echolume requires access to an **audio input source** (microphone, audio interface, virtual loopback device, or OSC network loopback) to drive visual frequencies. All audio capture, envelope tracking, and FFT analysis are executed **entirely on your local machine** within the app's execution context. We do not store, record, serialize, or transmit any audio files or inputs. Audio data stays in memory and is discarded immediately after visual frame calculation.

Diagnostic and crash reporting

Echolume uses no third-party crash-reporting or analytics SDK. Crash reports are handled entirely by Apple's built-in tooling: if you have opted into sharing analytics with developers in your macOS privacy settings, anonymized crash diagnostics for App Store and TestFlight builds reach us through Apple's Xcode Organizer and App Store Connect. This data is governed by Apple's Privacy Policy and contains no audio streams, personal names, account identifiers, or user files.

Twitch Chat Integration data handling

When you enable Twitch Chat integration, Echolume establishes a secure WebSocket connection directly to the public Twitch IRC servers (wss://irc-ws.chat.twitch.tv:443). The connection is **anonymous and read-only** (registering as a guest justinfan* account). Echolume processes incoming chat commands locally in memory to change themes or knobs. We do not log, store, or upload chat messages, user handles, or channel logs.

Website hosting & analytics

This marketing website is hosted using GitHub Pages. GitHub may capture standard HTTP access logs (including visitor IP addresses, browser user-agent tokens, and request timestamps) to maintain network stability and security, as outlined in the GitHub Privacy Statement. We do not use marketing or advertising trackers, and we set no cookies on this website.

Analytics

We use Umami, a privacy-friendly, self-hosted, cookieless analytics tool, to understand aggregate traffic such as page views, referrers, browser, and country. Umami does not use cookies, does not collect or store personally identifiable information, and does not track you across other websites. No data is ever sold or shared with advertisers.

Contact

If you have any questions regarding Echolume's security policies or local processing details, please check our Support Page for contact directions.
