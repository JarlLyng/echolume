How it works

Get Echolume

Built for performance

  • Input

    Choose any audio input device in-app — interface, loopback, or mic — without changing the system default.

  • Themes

    Pick a visual theme and shape style, tweak abstraction and motion, or hit Randomize and go.

  • Fullscreen

    One tap to fullscreen on your main display or an external output. Minimal overlay, maximum impact.

  • Low latency

    Stable, real-time audio analysis and Metal rendering. No streaming, no cloud — everything runs locally.
